# Tipping - Checkout Tips and Donations for WooCommerce

Let customers add an optional tip or donation at the WooCommerce cart and
checkout. Offer preset amounts (fixed or a percentage of the cart) plus an
optional custom amount. The tip is applied as a native cart fee, shown in the
totals and recorded on the order, and updates live as the customer selects it.

## Features

- Preset tip amounts as fixed values or a percentage of the cart total.
- Optional custom-amount field.
- Applied as a native WooCommerce cart fee, shown in totals and on the order.
- Updates live on selection via WooCommerce checkout AJAX.
- Customer-facing label and description you control.
- Show on the cart, the checkout, or both; optional taxable toggle (off by default).

## Installation

1. Upload the plugin to `/wp-content/plugins/tipping`, or install it via Plugins → Add New.
2. Activate it. WooCommerce must be installed and active.
3. Configure presets, label and placement under WooCommerce → Tipping.

## Frequently Asked Questions

**Does it require WooCommerce?**
Yes. WooCommerce must be installed and active.

**Is the tip taxable?**
By default, no. You can enable a taxable toggle under WooCommerce → Tipping.
