# Tipping - Checkout Tips and Donations for WooCommerce

Let customers add an optional tip or donation at the WooCommerce checkout. Offer
preset amounts — fixed or a percentage of the cart. The tip is applied as a
native cart fee, shown in the totals and recorded on the order, and updates live
as the customer selects it.

## Features

- Preset tip amounts as fixed values or a percentage of the cart total.
- Applied as a native WooCommerce cart fee, shown in totals and on the order.
- Updates live on selection via WooCommerce checkout AJAX.
- Customer-facing label and description you control.
- Fully opt-in: the default is "No tip", and tips are never taxed.

## Installation

1. Upload the plugin to `/wp-content/plugins/tipping`, or install it via Plugins → Add New.
2. Activate it. WooCommerce must be installed and active.
3. Configure presets and label under WooCommerce → Tipping.

## Frequently Asked Questions

**Does it require WooCommerce?**
Yes. WooCommerce must be installed and active.

**Is the tip taxable?**
No. Tips and donations are added as a non-taxable fee.
